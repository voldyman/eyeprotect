iProtect
--------
A Gtk+ application that uses the 20-20-20 rule to protect your eyes from damage.


##Build

Depencies:

* gtk+-3.0
* granite
* appindicator3-0.1

To build:

    $ git clone git@github.com:voldyman/iprotect.git
    $ cd iprotect
    $ mkdir build
    $ cmake .. && make
    $ ./iprotect
